# US1 Screenshot Checklist — Static Captions on All 6 Cards

**Goal**: Confirm every plot card shows a caption when the report is opened.

## Screenshots to capture (save to `us1/`)

| Card | File name | Acceptance criteria |
|------|-----------|---------------------|
| ACT Plot 1 — All Sessions (default) | `us1-act-plot1-all.png` | Caption text visible below legend; prose present; at least one N annotation visible in semi-bold |
| ACT Plot 2 — Heatmap | `us1-act-plot2.png` | Caption text visible below gradient legend; prose describes the plot |
| ACT Plot 3 — Radial Clock | `us1-act-plot3.png` | Caption text visible below legend; sleep-hours SD explanation present in prose |
| HR Plot 1 — Zone Time Allocation | `us1-hr-plot1.png` | Caption text visible below bar chart legend; at least two N annotations visible |
| HR Plot 2 — TRIMP (default) | `us1-hr-plot2-trimp.png` | Caption text visible below line chart; TRIMP prose present |
| HR Plot 3 — Adherence (default) | `us1-hr-plot3-adherence.png` | Caption text visible below heatmap legend; 75% threshold disclaimer present |

## Common failure modes to check

- Caption text overflows the card bottom edge
- Caption text overlaps the plot legend
- Numeric annotation label and value appear indistinguishable (same style)
- Prose text is cut off mid-word
